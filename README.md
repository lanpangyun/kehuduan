使用方法
如果您设备上运行了clash、v2rayN、其他机场客户端应用程序，请您先关闭并完全退出，避免代理设置干扰。

操作步骤：

1.下载相应操作系统的客户端安装文件，安装后登录账号

2.主页选择节点，选择模式智能分流或全局代理

👉模式说明：

智能分流：仅海外站点走代理

全局代理：所有站点走代理

3.点击未连接，开启代理，变成已连接

常见问题与解决
电脑客户端TUN模式（可选项，非必要步骤）
TUN模式也称为虚拟网卡模式，开启后所有流量都会被接管

👉什么时候才需要勾选TUN模式

电脑软件外网需求：除了浏览器，其他电脑软件也需要使用代理上外网
普通代理模式不生效：普通模式开启代理后，浏览器始终无法上外网
👉如果仅是浏览器需要访问外网，建议不勾选TUN模式

👉打开TUN模式方法：电脑完全关闭所有代理软件和本软件，托盘也退出。然后桌面找到蓝胖云应用图标，右键选择管理员运行，再勾选TUN模式。
